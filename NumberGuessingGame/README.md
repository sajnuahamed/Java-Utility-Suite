# Number Guessing Game

A simple console-based Java game where the player tries to guess a randomly generated number between **1 and 10**. The program provides feedback after each guess, indicating whether the guess is too high or too low, and displays the total number of attempts once the correct number is guessed.

## Features

* Randomly generates a number between **1 and 10**
* Accepts user input through the console
* Provides hints for incorrect guesses

  * **Too High**
  * **Too Low**
* Tracks and displays the total number of attempts
* Uses a `do-while` loop to ensure at least one guess
* Implements `try-with-resources` for proper resource management

## Technologies Used

* Java
* `Scanner` for user input
* `Random` for number generation

## Sample Output

```text
Guess a number from 1 to 10: 5
Oops! Too High! Try again!!

Guess a number from 1 to 10: 2
Oops! Too Low! Try again!!

Guess a number from 1 to 10: 4

🎉 Congratulations! You guessed the number: 4
Attempts: 3
```
