# Calculator

A console-based calculator built using Core Java that performs basic arithmetic and exponential operations based on user input.

## Features

* Addition (`+`)
* Subtraction (`-`)
* Multiplication (`*`)
* Division (`/`)
* Modulus (`%`)
* Exponentiation (`^`)
* Division-by-zero validation
* Formatted output

## Concepts Demonstrated

* User input using `Scanner`
* Enhanced `switch` expressions
* Arithmetic operators
* Ternary operator
* Conditional statements
* `Math.pow()` method
* Formatted output using `printf`

## Sample Output

```text
Enter the first number: 12
Enter the operator (+, -, *, /, %, ^): *
Enter the second number: 8
Result: 96.00
```

## How to Run

1. Compile `Calculator.java`.
2. Run the program.
3. Enter two numbers and select an operator when prompted.
