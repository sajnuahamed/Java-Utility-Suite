\# Dice Rolling Game



A console-based Java game that simulates rolling multiple dice. The user can enter the number of dice to roll, and the program generates random values for each die, displays the corresponding dice faces using ASCII art, and calculates the total score.



\## Features



\* Allows the user to roll multiple dice

\* Generates random dice values between 1 and 6

\* Displays visual representations of each dice roll

\* Calculates and displays the total value of all dice rolls

\* Validates that the number of dice entered is positive

\* Uses methods to organize dice display logic



\## Technologies Used



\* Java

\* `Random` for dice value generation

\* `Scanner` for user input

\* Text blocks for ASCII art representation

\* Enhanced `switch` expressions
